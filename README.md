# Advanced-todo-list
[![Synergia](https://img.shields.io/badge/Synergia-Boilerplate-green?style=flat)](https://www.synergia.dcc.ufmg.br)
[![Meteor-Framework](https://img.shields.io/badge/Meteor-Framework-red?style=flat&logo=meteor&logoColor=white)](https://www.meteor.com/)
[![React-Library](https://img.shields.io/badge/React-Library-61DAFB?style=flat&logo=react&logoColor=white)](https://react.dev/)
[![MUI-UI Library](https://img.shields.io/badge/MaterialUI-UI%20Library-blue?style=flat&logo=mui&logoColor=white)](https://mui.com/)

This project implements a task management web app, in which users can create an account, edit their profile, and create, view, update, and remove Tasks. Each task can be private or public; editing is only allowed by the creator.

The system offers two main areas:

- Welcome Screen: Displays a summary of the user's latest activity, featuring the "Atividades recentes" section with the 5 most recently added or updated tasks.

- Task list: gathers all public and private tasks of the logged-in user. In this view, it is possible to change the task's status (to do, in progress, or completed), as well as view, edit, or remove it. Additionally, it is possible to view other users' public tasks.


## Quick Start

```bash
# Clone the project
git clone https://github.com/iago-r/advanced-todo-list-boilerplate-version.git

# Enter the directory
cd advanced-todo-list-boilerplate-version

# Install the dependencies
meteor npm install

# Start the server
meteor run
```
